# Платформа управления проектной документацией

Централизованная система для нормализации документооборота в инженерных и ПИР-проектах.

## Стек
- Backend: FastAPI (Python)
- Frontend: React + TypeScript + Vite + TanStack Query
- БД: PostgreSQL
- Хранение файлов: MinIO / локальное хранилище
- Авторизация: JWT + HTTP-only cookies
